# NxtStep 🚀

NxtStep is an AI-powered career guidance application designed to help students explore career opportunities and make informed decisions about their future.

The application provides information about different career paths, including career descriptions, required skills, education pathways, salary information, and career categories. Users can browse careers, search for specific career options, bookmark careers, and explore career recommendations through an interactive quiz.

NxtStep also includes **Sage**, an AI Career Counselor that helps students ask questions about careers, skills, salaries, certifications, higher studies, and career planning.

## ✨ Features

- 🔐 User registration and login
- 🔎 Search and explore different careers
- 🗂️ Career categories such as Traditional, Emerging Tech, Science, Creative, and Underrated but Growing
- 📚 Detailed career information
- 🔖 Bookmark careers for future reference
- 🧠 Career exploration quiz
- 🤖 Sage AI Career Counselor
- 💬 AI-powered career guidance and recommendations
- 🌐 Web application deployment
- 📱 Android APK support

## 🚀 How to Run the Project Locally

### Prerequisites

Make sure the following are installed:

* Flutter SDK
* Dart SDK
* Android Studio (for Android development)
* Git

### Steps

1. Clone the repository:
git clone https://github.com/Ddivya11/NxtStep.git


2. Navigate to the project directory:
cd NxtStep


3. Install the project dependencies:
flutter pub get


4. Run the application:
flutter run


To run the web version:
flutter run -d chrome


To build the Android APK:
flutter build apk --release

The generated APK can be found at:

text
build/app/outputs/flutter-apk/app-release.apk
